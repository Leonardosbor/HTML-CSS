Principais formatações pt. 2

Texto marcado → MARK

Podemos criar também um marca texto usando a tag MARK → SORVETE SABOR ZAP

Texto grande e pequeno → big e small

a tag big ainda funciona, mas é obsoleta, por isso ela fica vermlehavermelha. Por outro lado, a tag small funciona perfeitamente

Texto deletado → del

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto inserido → ins

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois

Além da tag ins, temos também a tag não-semântica U → exemplo

Texto sobrescrito → sup

x20-3 = 0

Texto subscrito → sub
